# Ecommerce-Sales-Dashboard-Excel

Sales Performance Dashboard – Excel Data Analytics Project

This project presents an interactive Sales Performance Dashboard built in Microsoft Excel to analyze sales performance, customer behavior, and product trends.

The dashboard provides a comprehensive overview of business performance through key metrics, interactive slicers, and visual analytics.

Key Metrics

Total Orders: 2400

Total Quantity Sold: 11,997

Total Revenue: $64K

Average Customer Rating: 3.96

Average Delivery Days: 2.34

Average Order Value: 270.42

Average Selling Price: $54.10

Dashboard Features

Weekly revenue trend analysis

Sales breakdown by order mode (App, Website, Instagram, etc.)

Top-performing products analysis

Customer rating distribution

Revenue segmentation by gender

Geographic revenue visualization

Tools Used

Microsoft Excel

Pivot Tables & Pivot Charts

Data Cleaning & Transformation

Interactive Slicers

Dashboard Design


The objective of this project is to demonstrate how Excel can be used to analyze business data, track sales performance, and build interactive dashboards for decision-making.
